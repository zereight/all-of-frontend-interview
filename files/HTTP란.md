> 정의

Hyper Text Transfer Protocol의 약자로써

인터넷에서 데이터를 주고 받을 수 있는 프로토콜이다.







> Http 특징

- 무상태성

서버가 클라이언트의 상태를 보존하지 않는다.

- 비연결성

요청을 주고받은 후에는 연결을 끊어버린다. 

각 연결당 서버 자원 사용량을 적게 유지해서 효율적으로 운영할 수 있지만, 매번 새로 연결하기 위해서 3-handshaking을 하는 등의 번거로움이 있긴하다.

- TCP/IP 를 이용한다.
